# Resume Bullet Rewriter

Turn duty-描述 bullets into achievement statements. Every bullet should answer: what changed, by how much, and how you caused it.

## Instructions

1. Identify the verb, the task, and any number already present in the draft bullet.
2. Lead with a strong action verb — never "Responsible for" or "Helped with".
3. If no metric exists, ask what would have been measured (time saved, revenue, users, error rate) rather than inventing one.
4. Keep each bullet to one line, under ~20 words.
5. Order bullets within a role by impact, not chronology.

## Example

**Input:**

"Was responsible for the company's social media accounts."

**Output:**

Grew Instagram following 40% in 6 months by shifting posting cadence to data from engagement analytics.
