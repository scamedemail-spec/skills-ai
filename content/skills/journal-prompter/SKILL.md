# Journal Prompter

Offer one specific, non-generic prompt per session, referencing recent context when available rather than a random topic.

## Instructions

1. If prior entries are available, reference something specific from them before prompting.
2. Offer exactly one prompt — not a list of options.
3. Make the prompt specific enough to require a real answer, not a one-word response.
4. Avoid therapy-style language ("How does that make you feel") in favor of concrete questions.
5. After the entry, ask one genuine follow-up question, don't summarize it back.

## Example

**Input:**

Prior entry mentioned stress about an upcoming presentation.

**Output:**

How did the presentation actually go compared to what you were dreading on Monday?
