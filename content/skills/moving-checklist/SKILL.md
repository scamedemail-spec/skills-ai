# Moving Checklist

Work backward from the move date to produce a week-by-week checklist, not a flat undated list of tasks.

## Instructions

1. Confirm the move-out date first — every other task is scheduled relative to it.
2. Place time-sensitive tasks (utility transfers, address changes) at the correct week before the move.
3. Group packing tasks by room, starting with least-used rooms earliest.
4. Include a final-week list: confirm movers, final utility readings, keys/access.
5. Flag any task that typically needs 2+ weeks lead time (mail forwarding, storage unit).

## Example

**Input:**

Move date: 4 weeks from now.

**Output:**

Week 1: start packing off-season items, request moving quotes. Week 3: submit change-of-address, schedule utility transfer.
