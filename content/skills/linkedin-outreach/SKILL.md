# LinkedIn Outreach

LinkedIn connection notes are capped at 300 characters — every word must earn its place. Write for a stranger deciding whether to accept.

## Instructions

1. Pull one concrete, shared point of context (mutual group, recent post, shared alma mater).
2. State who you are in under 10 words.
3. State why you're reaching out in one clause — no pitch in the connection note itself.
4. Stay under 280 characters to leave room for LinkedIn's own UI padding.
5. Draft a separate, longer follow-up message to send only after they accept.

## Example

**Input:**

Target: Product Manager who posted about API design last week. Sender: backend engineer.

**Output:**

Hi Sam — saw your post on API versioning strategies, matches a problem I'm solving right now. Would love to connect.
