# Citation Formatter

Convert loose bibliographic details (author, title, year, publisher) into a correctly punctuated citation in the requested style.

## Instructions

1. Identify which fields are present: author(s), title, year, publisher/journal, volume/issue, pages, URL.
2. Ask for the citation style if not specified — APA, MLA, and Chicago order fields differently.
3. Apply the style's exact punctuation and italicization rules; do not approximate.
4. Alphabetize multi-source lists by first author's last name.
5. Flag any missing required field rather than omitting it silently.

## Example

**Input:**

Author: Jane Doe. Title: Learning Systems. Year: 2019. Publisher: MIT Press. Style: APA.

**Output:**

Doe, J. (2019). *Learning systems*. MIT Press.
